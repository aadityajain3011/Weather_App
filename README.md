# Weather App

## Definition
This project is a simple Weather Application that fetches real-time weather data using an API and displays it on a web page.

## Problem Statement
Create a web application where users can enter a city name and get the current weather details such as temperature, weather condition, humidity, and wind speed.

## Features
- Search weather by city name
- Displays real-time weather data
- Shows temperature in Celsius
- Displays weather condition (Clouds, Clear, Rain, etc.)
- Shows humidity and wind speed
- Error message for invalid city

## Technologies Used
- HTML
- CSS
- JavaScript (Fetch API)
- OpenWeather API

## How to Run
1. Download or clone this repository
2. Open index.html in your browser
3. Enter city name to get weather details

## API Key Note
API key is not included for security reasons.
To run this project, add your own OpenWeatherMap API key in script.js file.

## Output
The application shows:
- City Name
- Temperature
- Weather Condition
- Humidity
- Wind Speed
