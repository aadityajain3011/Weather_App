# Weather App

## Definition
This project is a dynamic Weather Application that fetches real-time weather data using the OpenWeatherMap API.

## Objective
The objective of this project is to allow users to search for a city and view its current weather details in a clean and user-friendly interface.

## Features
- Search weather by city name
- Displays:
  - City Name
  - Temperature
  - Weather Condition
  - Humidity
  - Wind Speed
- Real-time data using API
- Error handling (invalid city / empty input)
- Weather icons display
- Enter key support

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
- Fetch API

## Output
The application shows real-time weather details of any searched city.
